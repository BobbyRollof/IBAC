# IBAC PoC

## Description
This is a Proof of Concept (PoC) for Intent-Based Access Control (IBAC) using AI agents.

## Getting Started

Install Uvicorn with pip:
```bash
pip install fastapi uvicorn
```

Install Uvicorn with pip3:
```bash
pip3 install fastapi uvicorn
```

## Run the application with Uvicorn:
```bash
uvicorn ibac_poc:app --reload
```
